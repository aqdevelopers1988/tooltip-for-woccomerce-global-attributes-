Adds configurable storefront tooltips to WooCommerce global product attributes.

== Description ==

Tooltip for WooCommerce Global Attributes adds a tooltip text field to each WooCommerce global attribute in Products > Attributes. The tooltip is displayed next to the attribute label on the storefront.

A WooCommerce > Attribute Tooltips settings page lets store managers customize:

* Tooltip background color.
* Tooltip text color.
* Default icon color.
* Default icon border color.
* A custom uploaded tooltip icon.

== Installation ==

1. Upload this plugin folder to `/wp-content/plugins/`.
2. Activate the plugin in WordPress.
3. Go to Products > Attributes and add tooltip text to any global attribute.
4. Go to WooCommerce > Attribute Tooltips to customize colors and upload an icon.
